# Friday the 13th API

This API returns JSON whether today is Friday the 13th.

## Start up

```shell
$ deno run --allow-net main.ts
```

## usage

1. start app
2. access to <http://localhost:3000/>

## How to install `deno`

See [Deno manual](https://docs.deno.com/runtime/manual/getting_started/installation)
